# Keyboard Shortcuts

## General
| Key | Action |
|-----|--------|
| Ctrl+N | New schematic |
| Ctrl+Shift+N | New FluxScript tab |
| Ctrl+O | Open schematic |
| Ctrl+S | Save schematic |
| Ctrl+W | Close tab |
| Ctrl+Q | Quit |
| F1 | Open user manual |

## Edit
| Key | Action |
|-----|--------|
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |
| Ctrl+X | Cut |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+A | Select all |
| Ctrl+F | Find & replace |
| Ctrl+E | Batch edit properties |
| Del | Delete selected |

## Schematic Editor
| Key | Action |
|-----|--------|
| W | Wire tool |
| R | Rotate selected component |
| X | Flip horizontal |
| Space | Component selector |

## View
| Key | Action |
|-----|--------|
| Ctrl++ | Zoom in |
| Ctrl+- | Zoom out |
| Ctrl+0 | Fit to view |

## Simulation
| Key | Action |
|-----|--------|
| F5 | Run simulation |
| F6 | Stop simulation |
| Ctrl+P | Quick open / Place probe |

## Tabs
| Key | Action |
|-----|--------|
| Ctrl+Tab | Next tab |
| Ctrl+Shift+Tab | Previous tab |
| Ctrl+Shift+D | Duplicate tab |
| Ctrl+Shift+T | Reopen closed tab |
| Alt+1..9 | Jump to tab N |
