# Keyboard Shortcuts

Master VioSpice using these productive shortcuts.

## General
* **F1**: Open User Manual
* **Ctrl+O**: Open Project
* **Ctrl+S**: Save Project
* **Ctrl+Q**: Quit Application

## Schematic Editor
* **W**: Start Wiring tool
* **R**: Rotate selected component
* **X**: Flip component horizontally
* **Del**: Delete selected item
* **Space**: Open component selector

## Simulation
* **F5**: Run Simulation
* **F6**: Stop Simulation
* **Ctrl+P**: Place Probe
